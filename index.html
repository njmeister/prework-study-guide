HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Commands include:
    • h1-h6 for Headers
    • ul for an Unordered List
    • li for the List's Entries
    • section to create a new Section
  • Commands are closed using the same code but with a / preceding it.
  • CSS links go in the head.
  • JavaScript goes at the bottom of the body.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout: switches to a different branch.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds the changes made to the working branch.
  • git commit -m "(message)": commits the files to the repo with the message.
  • git pull origin main: pulls the main branch from Github to the local computer.
  • git push origin (branch name): uploads branch to the Github repo.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • By adding brackets, variables can contain an array of information separated by commas.
  • Control flow is the order in which a computer executes code in a script.
  • Functions define a section of code that can be executed when called upon instead of in the normal control flow.
  • console.log displays the input in the DevTools console.
